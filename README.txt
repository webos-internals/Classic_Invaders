
Classic Invaders is an alien shooter game written by Todd Steinackle. 

To play launch the executable and navigate the menu with the tab or up/down 
arrow keys then select with the space bar or enter key. The esc key can be 
used to exit any screen and the program itself.

The source code for Classic Invaders is released under a BSD license and is
available at http://www.noquarterarcade.com/.

The source can be built on Windows, Mac OS X, and Linux with no modifications. 
The SDL, SDL_image, SDL_mixer, SDL_ttf, and boost libraries are needed.
